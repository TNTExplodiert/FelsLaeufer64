Browser Jump and Run

Felslaeufer 64

Ein statisches 3D-Jump-and-Run mit Doppelsprung, Mauskamera und ASCII-Level-Editor. `S` setzt den Start, `L` ist Boden, `P` eine Plattform fuer einen normalen Sprung, `D` eine hohe Plattform fuer den Doppelsprung, `X` ein Loch und `Z` das Ziel.

Level-ASCII

64 x 64

Jede Zeile beschreibt eine Reihe des Spielfelds. Unerlaubte Zeichen werden automatisch zu `X` normalisiert.

Beispiel-Level geladen.

Steuerung

Live
  • WASD laufen relativ zur Kamerarichtung
  • Leertaste Sprung plus Doppelsprung
  • Q kurzer Dash etwa ein Feld weit
  • Maus Kamera drehen nach Klick ins Spielfeld
  • Mausrad Zoom
  • R Respawn am Start
  • ESC Maus freigeben
